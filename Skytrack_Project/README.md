# 🚁 SkyTrack Logistics - Dashboard de Drones

## 📌 Sobre o Projeto

Este projeto consiste em um protótipo de sistema logístico para gestão de drones, desenvolvido como atividade prática de Front-End.

A aplicação simula um dashboard operacional utilizado para monitoramento da frota, visualização de alertas e cadastro de novos drones.

---

## 🛠️ Tecnologias Utilizadas

* HTML5 Semântico
* CSS3 (Flexbox, Gradientes, Box Model)
* Design Responsivo (Media Queries)
* Git & GitHub (Versionamento)

---

## 🎯 Funcionalidades

* Visualização da frota de drones em tabela
* Exibição de status operacionais
* Cards de alertas (manutenção, bateria, prioridade)
* Cadastro de novos drones com validação
* Navegação entre páginas (Dashboard ↔ Cadastro)
* Botão SOS fixo na interface

---

## 💡 Desafios Superados

* Implementação da **Tríade do Truncamento** (ellipsis) para textos longos
* Uso de **Flexbox** para organização responsiva dos elementos
* Criação de formulário com **validação nativa do HTML5**
* Aplicação de **semântica HTML** para melhor estrutura e acessibilidade
* Adaptação do layout para diferentes tamanhos de tela

---

## 📱 Responsividade

O sistema foi desenvolvido para funcionar em:

* Desktops
* Tablets (principal foco do projeto)
* Telas menores com ajustes via media query (768px)

---

## 🚀 Como Executar

1. Baixe ou clone o repositório
2. Abra a pasta do projeto
3. Execute o arquivo `index.html` em seu navegador
